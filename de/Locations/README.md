# Orte

Nutze diesen Ordner fuer reale Orte in Bern, die in der Session vorkommen.

- Polizeiliche Orte
- Konzert- und Veranstaltungsorte
- Ermittlungsorte
- Verfolgungsrouten und oeffentliche Wahrzeichen

Jede Ortsdatei sollte festhalten:

- Warum der Ort fuer den Fall wichtig ist
- Welche Tonlage oder Szenenart er unterstuetzt
- Welche Hinweise, NPCs oder Vorfaelle dort stattfinden koennen
