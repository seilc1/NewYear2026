# Spielercharaktere

Diese Dateien liefern die vier Spielercharakter-Vorlagen fuer das improvisierte `B team` der Berner Polizei.

- Die Spielercharaktere sind Rookies aus unterschiedlichen Abteilungen
- Jeder startet mit vollem Namen, Abteilung und Abteilungsleitung
- Jede Vorlage laesst Raum dafuer, dass der Spieler persoenliche Vorgeschichte, genaues Aussehen und tiefere Charakterzuege selbst festlegt
- Die Grundidee ist, dass jeder Rookie eine praktische Kompetenz mitbringt, die beim Loesen des Falls um `The Mercenary` helfen kann

Aktuelle Vorlagen fuer Spielercharaktere:

- `Luca Baumann.md` - Patrouillenrookie unter `Carey Mahoney`
- `Nina Frei.md` - Verkehrsrookie unter `Debbie Callahan`
- `Timon Keller.md` - Rookie fuer Akten und Beweismittel unter `Larvell Jones`
- `Samira Aebi.md` - Forensikrookie unter `Moses Hightower`
