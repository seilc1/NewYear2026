# Verbrechen

Dieser Ordner enthaelt waehlbare Verbrechensziele fuer `The Mercenary`.

Jede Datei repraesentiert einen moeglichen Hauptplot fuer die Session.

- Waehle eines als echten Plan
- Nutze die anderen als falsche Faehrten, Backup-Ideen oder alternative Kampagnenversionen
- Halte jedes Verbrechen an die Berner Altstadt und den Silvesterdruck des 31.12.2026 gebunden
- Fuehre jedes Verbrechen durch dieselbe 4-Akt-Struktur, damit das genaue Ziel erst nach Akt 3 klar wird
- Nutze `Story Elements/Shared Clue Framework.md`, damit Akt 1 bis 3 ueber mehrere Verbrechensoptionen hinweg mehrdeutig bleiben
