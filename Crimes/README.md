# Crimes

This folder contains selectable crime objectives for `The Mercenary`.

Each file represents one possible main plot for the session.

- Pick one as the real plan
- Use the others as red herrings, backup ideas, or alternate campaign versions
- Keep every crime tied to Bern Old Town and the New Year's Eve pressure of 31.12.2026
- Run each crime through the same 4-act structure so the exact target is only clear after Act 3
- Use `Story Elements/Shared Clue Framework.md` to keep Acts 1 to 3 ambiguous across multiple crime options
